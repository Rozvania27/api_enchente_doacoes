# 📦 DoaCerta API

API backend para organização e otimização da distribuição de doações em cenários de enchentes.

---

## 📌 Sobre o Problema

O projeto aborda a má distribuição de doações´´ durante situações de emergência.

Em muitos casos, pontos de coleta recebem excesso de itens como roupas, enquanto outras regiões enfrentam escassez de recursos essenciais, como água e alimentos.

Esse desequilíbrio impacta diretamente a população afetada e reduz a eficiência da logística humanitária.

---

## 🧠 Solução Proposta

Desenvolvimento de uma API REST para:

- Cadastrar doações
- Gerenciar pontos de coleta
- Controlar categorias de itens (roupas, alimentos, água etc.)
- Simular redistribuição mais eficiente dos recursos

---

## 🏗️ Estrutura do Sistema

- Back-end
  
  API responsável por gerenciar os dados dos pontos de coleta e suas demandas.



- Banco de dados
  
 Armazena informações sobre locais e itens necessários, permitindo atualizações constantes.

---
 
## ⚙️ Tecnologias Utilizadas

- `Node.js`
- `Express.js`
- `VSCode`
- `Git & GitHub`
- `Postman`
- `Kaggle` (datasets para simulação de dados reais)
- `ChatGPT` (apoio no desenvolvimento, arquitetura e debugging)

---

## 🎯 Objetivo do Projeto

Criar uma base backend escalável para organizar, monitorar e otimizar a distribuição de doações, contribuindo para maior eficiência em situações de emergência.

---

## 📊 Dataset

Os dados utilizados para simulação foram baseados em datasets do Kaggle, representando cenários reais de doações e necessidades em situações de desastre.
